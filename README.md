# Task Management Dashboard (MERN)

A full-stack Kanban-style task manager with JWT auth, drag-and-drop, and live stats.

## Tech Stack
- **MongoDB** – database
- **Express** – REST API
- **React (Vite)** – frontend
- **Node.js** – runtime

## Setup

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)

### 1. Backend
```bash
cd server
npm install
# create .env file (see .env.example)
